Dimensión 3: Tecnología habilitante.

  1. Nuestro nivel de automatización y uso de tecnología de operación (OT) en procesos productivos.

    • Operación manual de todos los procesos productivos.
    • Parte de los procesos productivos ya están automatizados.
    • Operación automática de (casi) todos los procesos operativos.
    • Aún hay intervencion manual para ajuste, programación y/o soporte.
    • Procesos operativos al 100% automatizados.
    • Cada máquina, estación con configuración o diagnostico remoto.
    • Intervención humana solo para mantenimiento fisico.
    • Intervención manual solo para soporte.

  2. Nuestro nivel de conexión de sistemas operativas (OT) con sistemas de información (IT) y nivel de envío e uso de información de producción.

    • No se generan datos del proceso productivo.
    • Se generan datos mediantes sensores.
    • Se recopilan algunos datos del proceso de manera manual en papel, carpetas fisicas o MS Excel.
    • Casi todas las máquinas, estaciones generan datos mediante PLC, ESCADA, etc.
    • Estos datos estan extraidos manualmente, usando formularios digitales.
    • Todas las máquinas, estaciones envian datos automáticamente mediante IoT y en tiempo real para su análisis oportuno.

  3. Usamos datos para garantizar la ejecución y estabilidad del proceso productivo.

    • Sin monitoreo del proceso productivo en las maquinas, estaciones.
    • Monitoreo in situ por interfaz hombre máquina, estación.
    • Se generan algunos datos e indicadores del proceso.
    • Monitoreo in situ por interfaz hombre máquina, estación.
    • Indicadores con detección automática de fallas.
    • Monitoreo in situ y remoto de la máquina, estación.
    • Detección automática de fallas.
    • Notificaciones automáticas.
